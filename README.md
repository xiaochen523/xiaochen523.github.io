## 日常知识点2

#### C代码逻辑删除
```
#if 0
......
#endif
```
#### 一般来说人们习惯用注释符号'/*''*/'来消除代码，但是一但被注释部分有注释符号，'/*''*/'会就近选择注释符来完善本身，故绝大多数情况下删除会出错，所以人们开始使用#if来进行逻辑删除
